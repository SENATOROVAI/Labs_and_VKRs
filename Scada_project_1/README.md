# Лабораторная работа: интеграция ВЛ 6–10 кВ в SCADA-систему

**Тема:** Интеграция автоматизированного участка воздушной линии 6–10 кВ в SCADA-систему и разработка прототипа системы мониторинга с учётом рыночных тенденций.

## Состав проекта

```
Scada_project_1/
├── src/
│   └── scada_prototype.py       — демонстрационный прототип на Python
├── screenshots/
│   ├── architecture.png         — схема архитектуры системы
│   ├── flowchart.png            — блок-схема алгоритма прототипа
│   ├── 1.png                    — скриншот работы прототипа (штатный режим + авария)
│   └── 2.png                    — скриншот журнала событий (провал напряжения)
├── report/
│   ├── lab_report_ITMO.docx     — финальный отчёт (оформление по ИТМО)
│
└── README.md
```

## Запуск прототипа

```bash
python3 src/scada_prototype.py
```

Прототип имитирует 10 циклов опроса трёх секций линии. По завершении выводит журнал событий и статистику обнаруженных аварий.

## Ключевые параметры системы

| Параметр | Значение |
|---------|---------|
| Напряжение линии | 6–10 кВ |
| Уставка предупреждения по току | 350 А |
| Уставка аварии по току | 400 А |
| Минимальное напряжение | 5,5 кВ |
| Максимальное напряжение | 11,0 кВ |
| Период опроса | 1 с |
| Протокол связи RTU–SCADA | Modbus TCP / МЭК 60870-5-104 |
